# Romanian Writing Style Guide (inteles.ro)

- Diacritics mandatory: ă, î, â, ș, ț
- Tone: profesionist, empatic, clar; evită limbaj corporatist
- Paragrafe scurte: 2–3 propoziții
- Răspuns rapid (2–3 paragrafe inițiale)
- Structură recomandată: Intro → Perspectivă psihologică → Scenarii → Resurse → FAQ → Concluzie
- Evită clișee („a aprofunda”, „utiliza”), preferă exemple concrete
- Linkuri interne când relevant; external cu nofollow/sponsored
- Lungime tipică: 900–1200 cuvinte pentru draft inițial; 2000–2500 pentru final
