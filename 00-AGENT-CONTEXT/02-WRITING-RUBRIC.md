# Writing Rubric (Score Yourself 0–10)

1) Intent: Quick answer in first 2–3 paragraphs
2) Depth: Real insight (Jung/Freud or usage/mechanism)
3) Structure: H2/H3 every ~300–400 words; lists concise
4) Length: 2,000–2,500 words; no filler
5) Mobile: short paragraphs; clean spacing; no complex tables
6) Credibility: cite 1–2 reputable sources
7) FAQs: ≥6 Question/Answer blocks with schema
8) Images: 1–2 helpful visuals; strong alt text
9) Monetization: 1 (max 2) contextual affiliate links + disclosure
10) Language: diacritics; professional, empathetic; no AI‑tells

If any <7/10 → fix before publishing.
