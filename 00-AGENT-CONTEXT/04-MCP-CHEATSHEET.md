# MCP Cheatsheet (Ultra‑Lean)

WordPress
- List short: category 5 (<1000w), 50 per page, order=id desc
- Get post: ID → content + wordCount
- Update post: set content HTML, status publish
- Upload media: file + alt; attach to post

Pexels
- Search 5–8 landscape images for [topic]; pick 2–3 best
- Provide alt in Romanian; suggest placements

Monetization
