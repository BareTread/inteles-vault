# Style Guardrails (Do / Don’t)

Do
- Diacritice corecte; voce profesională, empatică
- Paragrafe scurte; liste; H2/H3 la ~300–400w
- 1–2 linkuri afiliate max; disclosure
- 1–2 imagini utile; alt în română
- Citează 1–2 surse credibile

Don’t
- „În concluzie, putem spune…”, „Este important de menționat…”, „Nu în ultimul rând…”
- Gradient/emoji‑spam; tabele complicate pe mobil
- Filler repetitiv sau „keyword stuffing”
- Limbaj presant/„ofertă limitată”
