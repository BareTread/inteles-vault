# TVA Ghid Complet 2025 - Fișier README

## Descrierea Articolului

Acest articol reprezintă un ghid complet și actualizat despre TVA (Taxa pe Valoarea Adăugată) în România, cu includerea modificărilor esențiale programate pentru 2025.

## Conținut Principal

### Puncte Cheie Acoperite:

1. **Definiție și principii TVA** - Explicații clare ale mecanismului de funcționare
2. **Cote actuale vs. cote 2025** - Comparare detaliată a modificărilor:
   - Cota standard: 19% → 21% (din august 2025)
   - Cote reduse: 9% și 5% → cotă unică de 11%

3. **Calcul TVA** - Formule și exemple practice pentru ambele perioade
4. **Obligații antreprenori** - Plafon înregistrare, declarații, termene
5. **Deductibilitate** - Condiții și limitări specifice
6. **Excepții și scutiri** - Complete și parțiale
7. **Comerț internațional** - Reguli pentru importuri, exporturi, tranzacții intracomunitare
8. **e-Factura** - Sistemul de facturare electronică
9. **Consecințe nerespectare** - Sancțiuni și măsuri de executare
10. **Pregătire 2025** - Adaptări necesare pentru tranziție
