# Project Summary — inteles.ro

Misiune: articole în limba română care explică sensuri (în special interpretări de vise) — clare, credibile, mobile‑first, cu monetizare echilibrată.

Public‑țintă: utilizatori mobili (97.5%), căutări rapide + interes pentru profunzime psihologică/culturală.

Principii:
- Răspuns rapid → profunzime (Jung/Freud) → scenarii → cultură → FAQ (6+)
- 1–2 linkuri afiliate în casetă verde (ANPC disclosure)
- Imagini Pexels relevante (alt corect), fără „decorații” inutile
