## Questions & Troubleshooting

**Q: How do I know which posts are high-traffic?**
A: Install Google Analytics or use WordPress plugins like MonsterInsights. Or check Google Search Console.

**Q: What if affiliate links break?**
A: Run monthly audit: "Check all affiliate links on inteles.ro"

**Q: Should I disclose on every post or site-wide?**
A: Both - site-wide in footer + per-post near each link (Romanian law requires per-link disclosure)

**Q: Can I use Amazon Associates for Romanian traffic?**
A: Yes, but Amazon.de or Amazon.it (not Amazon.com). Conversion is lower than local merchants like Libris/eMAG.

**Q: How long until I see revenue?**
A: 2Performant pays ~60 days after transaction. First earnings: Month 3. Consistent revenue: Month 6+.
