## Quality Standards

### Word Count
- **Minimum:** 2,000 words
- **Optimal:** 2,000-2,500 words
- **Maximum:** 3,000 words (avoid bloat)

**Why 2,000+:**
- Comprehensive coverage
- Better Google rankings
- More keyword opportunities
- Higher user value

### Content Depth

**Essential Elements:**
1. **Quick answer** (first 2-3 paragraphs)
2. **Psychological perspective** (Jung, Freud)
3. **Cultural context** (Romanian traditions)
4. **Practical application** (real-life scenarios)
5. **Scientific backing** (when applicable)
6. **FAQ section** (6 questions minimum)

**Avoid:**
- Bloated repetition
- Weak transitional text
- Filler content
- AI-obvious phrasing
- Manipulation tactics

## Writing Guidelines

**DO:**
- Professional and authoritative
- Empathetic and understanding
- Educational and informative
- Culturally aware (Romanian context)
- Evidence-based and credible

**DON'T:**
- Aggressive or pushy
- Manipulative or scammy
- Overly casual or unprofessional
- Sales-focused
- Clickbait-y

## Mobile Optimization

### Critical Stats
- **97.5% mobile traffic** on inteles.ro
- Mobile-first indexing by Google
- High bounce rate if not optimized

### Mobile-First Checklist

**Paragraph Length:**
- Max 3-4 sentences per paragraph
- Break long paragraphs into shorter ones
- White space is your friend

**Heading Hierarchy:**
```
âœ… GOOD:
H2 â†’ Short, scannable sections
H3 â†’ Bite-sized subsections
Frequent headers every 300-400 words

âŒ BAD:
Long walls of text
Rare headers
Dense paragraphs
```

**Lists & Bullets:**
- Use extensively for scannability
- Keep list items short
- Bold key terms in lists

**Images:**
- Optimize for mobile (max 1200px width)
- Fast loading (compressed JPEGs)
- Alt text for SEO
- Responsive sizing

**Tables:**
- Simple 2-3 column max
- Mobile-responsive HTML
- Avoid complex layouts
- Consider converting to lists on mobile

## Content Checklist

Before publishing, verify:

### Quality
- [ ] 2,000+ words of genuine value
- [ ] Credible sources cited (WHO, Jung, etc.)
- [ ] Professional, informational tone
- [ ] Comprehensive coverage of topic
- [ ] Evidence-based information

### SEO
- [ ] Natural keyword integration
- [ ] 6+ FAQ questions with schema
- [ ] Clean URL structure
- [ ] Meta description optimized
- [ ] Internal links to related articles

### Mobile
- [ ] Short paragraphs (3-4 sentences max)
- [ ] Frequent headers for scanning
- [ ] Lists and bullets used extensively
- [ ] Images compressed and responsive
- [ ] Fast loading (test on mobile)

### Monetization
- [ ] Max 1-2 contextual affiliate links
- [ ] Natural placement in resource box
- [ ] Helpful recommendation tone
- [ ] ANPC disclosure included
- [ ] No aggressive sales tactics

### Design
- [ ] Clean 4px borders (no gradients)
- [ ] Professional color palette
- [ ] Readable fonts and spacing
- [ ] No visual manipulation
- [ ] Consistent styling
