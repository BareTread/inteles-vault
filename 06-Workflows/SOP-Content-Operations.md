# SOP — Content Operations (inteles.ro)

Scop: proces cap‑coadă pentru upgrade/creare articole care satisfac intenția, cresc încrederea și monetizează discret.

1) Identifică articolul țintă
- Prioritizează: sub 1000w, fără FAQ, trafic bun/temă importantă.

2) Cercetare rapidă
- SERP + „People also ask”, Wikipedia‑RO, surse psihologie (Jung/Freud).

3) Outline pe șablon
- Vise: răspuns scurt → Jung/Freud → scenarii → tradiții → resurse → FAQ 6+
- „Ce înseamnă”: definiție → utilizare → exemple → resurse → FAQ 6+

4) Scriere & structurare
- 2.000–2.500w, paragrafe scurte, H2/H3 frecvente, liste clare.

5) Imagini (Pexels MCP)
- 1–2 imagini relevante (alt în română); plasează doar unde ajută.

6) Monetizare echilibrată
- 1 (max 2) linkuri în caseta verde; disclosure ANPC inclus.

7) Polish & publicare
- 1–2 linkuri interne + 1–2 citări externe; verifică diacritice; testează pe mobil; publică via MCP.

8) Tracking
- Actualizează pipeline + Master List (ID, titlu, înainte/după, % creștere, status).

Check final:
- [ ] Răspuns rapid în intro
- [ ] 2.000–2.500w
- [ ] 6+ FAQ (schema)
- [ ] 1–2 imagini utile (alt corect)
- [ ] 1–2 affiliate links (casetă verde + disclosure)
- [ ] Linkuri interne + surse credibile
- [ ] Test mobil (aspect + performanță percepută)
